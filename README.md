# API Documentation

Welcome to the API documentation for this Project. 

## Base URL

The base URL for all API endpoints is: `http://localhost:8000/api/products/`
Hosted Api Url: `https://productdashbackend.onrender.com/` <br/><br/>
while testing online Api replace `http://localhost:8000/` with `https://productdashbackend.onrender.com/` <br/> <br/>


1) Initilize database `http://localhost:8000/api/products/`

2) Transection Api `http://localhost:8000/api/products/transections?q={String}&month={String}&page={Number}`

3) Statistics Api `http://localhost:8000/api/products/statistics?&month=${String}`

4) Bar Chart `http://localhost:8000/api/products/barchart?&month=${String}`

5) Pie Chart `http://localhost:8000/api/products/piechart?&month=${String}`

6) Combine Results `http://localhost:8000/api/products/combineresults?&month=${month}`
